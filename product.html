    Colect API Integration

  • Easy to use API available if you want to create the connection yourself. For creating new integrations we advise you to use our Colect WebServices API. Our API has full support for synchronising products, customers, access rules, stock-levels, prices and more. Orders can either be requested or pushed back directly to your backoffice system. We have a full set of documentation available; please contact us to gain access to our API documentation.

    • CSV/XML Integration

  • Support for a number of file based integrations with automatic scheduling. Colect also supports a number of file based integrations (either XML or CSV) through a secure FTP interface. These can be scheduled to check for data-updates at regular time intervals.
